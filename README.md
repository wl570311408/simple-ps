# 小学生小报排版工具

一个专为小学生设计的小报排版工具，提供简单易用的图片、文字和形状编辑功能。

## 功能特性

### 图片编辑
- 拖拽移动图片位置
- 裁剪功能：支持设置上下左右边距
- 缩放调整：拖拽角保持比例，拖拽边自由调整
- 圆角处理：可设置圆角半径
- 层级管理：支持置顶、置底操作

### 文字编辑
- 添加自定义文字内容
- 字体选择：支持微软雅黑、黑体、宋体、楷体等
- 字号调整：12px - 72px
- 颜色设置：支持颜色选择器
- 对齐方式：左对齐、居中、右对齐
- 文字样式：粗体、斜体

### 形状工具
- 添加矩形和圆形形状
- 自定义形状颜色
- 边缘羽化效果
- 支持拖拽移动和缩放

### 画布设置
- 预设尺寸：A4、A5、A6、Letter、Legal、海报
- 自定义画布尺寸
- 背景设置：支持颜色和图片背景
- 画布缩放：自动适配窗口大小

### 其他功能
- 撤销/重做：支持历史操作记录
- 复制元素：快速复制选中的素材
- 删除元素：删除不需要的素材
- 键盘微调：使用方向键精确调整位置
- 导出功能：保存为 PNG 图片

## 技术栈

- Vue 3 + Script Setup
- Vite
- Tailwind CSS 3
- Lucide Vue Next (图标库)
- html2canvas (导出功能)

## 使用说明

1. **添加素材**：点击顶部工具栏的「添加图片」「添加文字」「添加形状」按钮
2. **选中素材**：点击画布上的素材进行选中
3. **编辑素材**：选中素材后，左侧面板会显示对应的编辑选项
4. **移动素材**：拖拽素材可以移动位置
5. **缩放素材**：拖拽素材边角的控制点进行缩放
6. **删除素材**：点击左侧面板的删除按钮或选中后按 Delete 键
7. **导出图片**：点击顶部工具栏的「导出图片」按钮

## 注意事项

- 素材可以部分超出画布边界，但不能完全在画布外
- 导出的图片尺寸与画布尺寸一致
- 建议使用现代浏览器（Chrome、Firefox、Edge）以获得最佳体验
